Socog Bot - a writing coach for Google Docs

To start using the bot:

  1. Open your Google Doc
  2. Mention @bot@socog.ai in a comment
  3. Give the bot permission to comment

Example Comments:

  • @bot@socog.ai Can you give me general feedback on my story?
  • @bot@socog.ai How can I make this argument more persuasive?
  • @bot@socog.ai Could you suggest some experiments my team could perform to test this hypothesis?
  • @bot@socog.ai What would be Alexander Hamilton's reaction to this argument? And how would Thomas Jefferson respond?
  • @bot@socog.ai What are some past businesses that faced similar issues in the past? How did they resolve them?
  • @bot@socog.ai Could you rewrite the job description section to be more attractive to ecologically minded grad students by emphasizing the impact on green technology development and the portability of open source software skills?

Tips:

  • To create a comment, select some text and click Insert > Comment.
  • Ctrl+Alt+M is the Windows keyboard shortcut to create a comment.
  • Cmd+Option+M is the Mac keyboard shortcut to create a comment.
  • You only need to select a few words of text, so the bot knows where you are looking.
  • Ask the bot for high-level advice.
  • The bot is very knowledgeable.
  • The bot is programmable: you can prompt it to perform complex editing tasks.
  • The bot can discuss content, add comments, and suggest changes.
  • If you want spelling and grammar help, try a better tool like Grammarly.
  • If you want content generation or rewriting, try a better tool like Google Gemini.
  • If you get annoying email notifications every time the bot comments, you can turn them off in Google Docs settings or directly in the email (see "Notification settings").
  • The bot is young and buggy. Please report any issues by typing BUG in a comment. Thank you for your patience!
